# Nagma Enfield Ltd - Website Build Evidence (nagmaenfield.co.uk)

This folder documents key configuration steps and backend setup evidence for the Nagma Enfield Ltd WordPress website, including pages setup, plugin configuration, membership roles, and appointment workflow management.

**Live site:** https://nagmaenfield.co.uk/

---

## What’s covered here
- Website page creation and structure
- Plugin installation and integration approach
- Membership roles configuration (access control / user roles)
- Appointment workflow setup and backend scheduling evidence
- Application form preview (recruitment / onboarding / enquiries)
